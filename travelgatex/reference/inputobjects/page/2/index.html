Filter that selects the filter criteria which will be used in this availability. Currently you can only choose the accesses. You must choose one of them, include or exclude, or the other alternative isn’t specified anything. If input both, you will receive a validation error that indicates this error. GraphQL Schema definition type FilterInput { access: AccessFilterInput rateRules: RateRulesFilterInput } Fields access(AccessFilterInput): You can specify one of the filters or any of them. »
